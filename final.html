Linguagens de Programação

Linguagem de programação é basicamente um conjunto de informações que são inseridas no computador e ele vai executá-las de acordo com a lógica que foi implementada, ou seja, a linguagem de programação é uma forma de traduzir comandos para que o computador possa entende-los e executá-los.

"Linguagem de Programação é uma linguagem escrita e formal que especifica um conjunto de instruções e regras usadas para gerar programas (software)."

As principais linguagens que rodam no lado do servidor são:

  • PHP
  • Javascript
  • Ruby
  • Java
  • C#

E do lado do client são:

  • React
  • Javascript
  • HTML:5*
  • CSS
  • Angunlar

É preciso observar que Javascript funciona tanto do lado Server quanto do lado Client, pois possui um subprograma presente nos navegadoresno lado do client e no Node Jsno lado server chamado V8, que faz a interpretação do Javascript para os mesmos.

*: O HTML:5 na realidade não é uma linguagem de programação, ele é uma linguagem de marcação com textos delimitados por nomes que o navegador consegue interpretar, onde cada um destes nomes possuí uma função e comportamento específico.

Como é uma estrutura básica de um HTML?

A estrutura básica do HTML é bem simples, primeiro ela começa com a tag "< html >", ela representa basicamente todo o arquivo, toda a página do site; depois vem a tag "< head >", que seria a cabeça da estrutura, responsável por fazer o carregamento de todas as informações antes de exibir para o usuário, a maioria das irfomações que estiverem dentro desta tag não aparece para o client, com excessão da tag "< title >", onde é inserido o título da página que foi aberta, todas informações ficam visiveis apenas para quem esta programando, ao final fecha-se a tag com < / head >; em seguida vem a tag "< body >" que é basicamente o corpo da estrutura, é o conteúdo da página, tudo que for inserido nela aparecerá diretamente para o client, ao final fecha-se a tag com < / body > e fecha-se a estrutura com < / html >
